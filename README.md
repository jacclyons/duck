# Duckkinesis: City Mayhem

Multiplayer telekinesis battles. You're a duck with telekinesisâ€”grab crates, cars, and cones, then throw them at other players.

## Run multiplayer

```bash
npm start
```

Then open http://localhost:3000 in multiple browser tabs (or different devices on the same network).

## Play offline

Open `duck.html` directly and click **Play Offline**, or run the server and click **Play (Multiplayer)** to play alone.

## Controls

- **WASD** move
- **Mouse** look
- **Space** jump
- **Shift** sprint
- **E** grab (telekinesis)
- **Wheel** pull/push held object
- **Left click** throw
- **Right click** drop
- **R** reset
