# Sound Effects Needed

Place these audio files (MP3 or WAV) in this folder:

| File | When it plays |
|------|----------------|
| `grab.mp3` | When you grab an object with E |
| `throw.mp3` | When you release a throw (left click) |
| `drop.mp3` | When you drop an object (right click) |
| `charge.mp3` | Optional: subtle loop while charging a throw |
| `jump.mp3` | When you jump (Space) |
| `shatter.mp3` | When a building breaks from impact |
| `impact.mp3` | When a thrown object hits something (building, NPC, ground) |
| `npc_hit.mp3` | When an NPC gets hit by a block or another duck |

---

**Minimum set** (if you want to start small):
- `grab.mp3`
- `throw.mp3`
- `shatter.mp3`
